Generative AI Chatbot Course Structure

Course Duration: 06 Month's

Building and Integrating Artificial Intelligence Chatbots and Voice Applications with Dialogflow and ChatGPT

HTML, CSS & JavaScript

In the first 1.5 months of our journey, we covered HTML, CSS, and JavaScript. We also did hands-on practice to solidify our understanding.

Teachable Machine

Then, we learned about a tool called Teachable Machine in 10-15 days. We created projects using it as well.

Backend

After that, we studied MongoDB for the backend. We learned how to use MongoDB to store and retrieve data.

Dialogflow

Then, we learned about Dialogflow, which is a major part of our course. We learned how to build conversational AI agents using Dialogflow.

Open AI

After that, we used OpenAI API keys to build chatbots. We learned how to generate text & translate languages using OpenAI's models.

Other's AI Tools

We then explored a variety of AI tools, such as Alan AI, Flowise AI, and more. We learned how to use these tools to solve real-world problems.
